ML: Prediction of Passengers Survival Rate in the Titanic Sinking

May 29, 2021

For this project I analyzed data from the passengers on board the Titanic. I created a Logistic Regression model that predicts which passengers survived the sinking of the ship, based on features like age and ticket class. The RMS Titanic set sail on its maiden voyage in 1912, crossing the Atlantic from Southampton, England, to New York City. The ship never completed the voyage, it sank to the bottom of the Atlantic Ocean after hitting an iceberg, bringing down 1,502 of 2,224 passengers on board.

Throughout the project you will be able to find:

  • Data cleaning and standardization.
  • Features and labels for model.
  • Logistic Regression Model.

Datasets and Technologies Used

The analysis was done with a csv file from Kaggle ‘titanic.csv’. It contains information of 1309 passengers on board the Titanic like their name, sex, age, the ticket class, ticket number, the cabin and the port they embarked.

In terms of technologies, python was used for the data analysis with the following tools: pandas for data tyding and manipulation, numpy for numerical analysis matplotlib and seaborn for plots and graphs, sklearn.preprocessing for normalizing features and sklearn.linear_model for the Logistic Regression model.

Data Analysis

Features and Labels for Model

The features selected for the model are: Sex, Age, FirstClass and SecondClass. In that sense, based on the passengers sex, age and ticket class the model is going to predict if the passenger was saved or it perished. Since the Logistic Regression model only accepts numbers as input, binning techniques were used on the previously mentioned features to convert them to numbers. The Sex feature had two values ‘male’ and ‘female’, ‘male’ was mapped to 0 and ‘female’ to 1. The FirstClass feature was created from Pclass (ticket class), if its value was 1, its value was mapped to 1, else to 0. The SecondClass was also created from Pclass, if its value was 2, its value was mapped to 1, else to 0. The table below shows the first 5 rows of the Features DataFrame.

On the other hand, the Labels DataFrame shows for each passenger if it survived or perish in the sinking of the ship. A 0 means that the passenger perished while a 1 means that he survived. The table below shows the first 2 rows of the Labels DataFrame.

Logistic Regression Model

Continuing with what was stated previously, the data is divided into train (80% of data) and test (remaining 20%) for the Features and Labels DataFrames. The training data is used to create the logistic model, the model learns from this data and finds the best intercept and coefficients in an iterative process. On the other hand, the test data has not been seen by the model and is fed to it to evaluate its accuracy. The following step is normalizing the features train and test datasets so that all its values are transformed in a [0 - 1] range.

Moving forward, a Logistic Regression model calculates the probability of a sample belonging to the positive class (passenger survived). With the formula Z = bo + b1X1 + … + bnXn, where b0 is the intercept and b1 … bn are the coefficients of each feature analyzed, the best intercept and coefficients are found. In that sense, using sklearn.linear_model the features_train and labels_train datasets are fed to the Logistic Regression model. The best intercept and coefficients for each feature are the following:

  • Intercept: -0.7092
  • Sex: 1.6721
  • Age: -0.3095
  • FirstClass: 0.8160
  • SecondClass: 0.3933

The Sex of a passenger, with a coefficient of 1.67, and the type of ticket being FirstClass, with a coefficient of 0.81, are the features that have the biggest influence in classifying a passenger as a survivor (positive class). Also the Test Score on the test data is 0.8740 this means, that on average 87.4% of the times the model will correctly classify a new passenger as surviving or perishing in the Titanic voyage.

The Confusion Matrix in Figure 1 shows the number of True Positives, True Negatives, False Positives and False Negatives. The number of True Positives is 74 that is the number of times the model predicted the passenger survived and he survived. There are 155 True Negatives that is the number of times the model predicted the passenger perished and he perished. On the other hand, there are 12 False Positives that is the number of times the model predicted the passenger survived ant he perished. Finally, the number of False Negatives is 21 that is the number of times the model predicted the passenger perished and he survived.
